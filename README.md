# Welcome to AAPI

**AAPI** (Automatic API) it's an automatic API builder. build with nodejs for create a nodejs api,
with the base endpoint build in.


# Installation

For use it you just need to install it by :
```bash
npm i @tashikomaa/aapi -g
```

then you go where you want to create your API's folder

## Create files and folders

For create your folder you just need to run :
```bash
aapi
```
